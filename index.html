Backstock Tracker App

I developed this application to manage and simplify the overstock process at my workplace with a few key goals. It provides tools to modify inventory forecast, display replinishment, and home replenishment.


Primary Project Objectives:

1) Improve data integrity
2) Improve workflow
3) Simplify Tasks
