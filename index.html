Backstock Tracker App ∧ ∨

I developed this application to manage and simplify the overstock process at my workplace with a few key goals. It provides tools to modify inventory forecast, make claims, display replinishment, and home replenishment.

Primary Project Objectives:

  • Improve data integrity
  • Improve workflow
  • Simplify Tasks
Article Lookup

This menu allows you to lookup all of the information about a given item in the database

Article Overide

This menu allows the user to change the SRQ (how much the shelf can hold) and the Case size (how many units in a case). altering these values changes how many items the system will think can go out.

Forecast Menu

This menu allows the user to see all forecast change requests

Forecast Details

Found by clicking on an item in the forecast menu. This menu allows the user to see all forecast request details

Forecast Form

Found by clicking the plus sign in the forecast menu. This menu allows the user to input a new forecast change request

Reversepick Menu

This menu tells the user what items can go out. It distinguishes a case from a single unit with "ea". It also says the storage location below the quantity to go for fast retrieval.
