Backstock Tracker App

I developed this application to manage and simplify the overstock process at my workplace with a few key goals. It provides tools to modify inventory forecast, make claims, display replinishment, and home replenishment.

Primary Project Objectives:

  • Improve data integrity
  • Improve workflow
  • Simplify Tasks
